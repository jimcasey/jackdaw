Jackdaw is a **funnel, not an archive**. Notes flow Capture → Triage → Export
and then leave the app. Defined in CLAUDE.md and `docs/prd/concept-brief.md`.

**Why:** The user (owner) already has a permanent notes home (Obsidian + GitHub
repo). The unmet need is a fast on-ramp, not another place notes pile up. Any
drift toward organizing/searching/browsing recreates the exact problem the app
exists to fix.

**How to apply:** This is the tie-breaker for most scope questions. If a
proposed feature helps fast capture or clean handoff → in scope. If it moves
toward organizing, searching, browsing, or retaining past notes → non-goal.
Use it before relitigating scope. See [[project_v1_scope]].
